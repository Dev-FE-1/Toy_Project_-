2. **관리자용 페이지**
    1. CSS 스타일이 적용된 직원 프로필 페이지 개발

        - querySelector, getElementsByClassName, createElement, appendChild, removeChild 등 사용
        
3. **사용자용 페이지**
    1. 마이 페이지 구현

            - 시간 관리 기능 개발
            - 현 시각을 표시하는 시계 (타이머) 구현
            - 토글 형태의 근무 시작 / 종료 스위치 구현
            - 모달을 활용한 근무 시작 / 종료 확인 창 구현
    2. 기업 공지 모음 갤러리 구현
